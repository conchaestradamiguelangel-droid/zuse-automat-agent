# Changelog

## v1.0 - 2026-06-04

Initial public release accompanying the Zenodo preprint:
[10.5281/zenodo.20516375](https://doi.org/10.5281/zenodo.20516375).

### Scientific Milestones

- Built a deterministic discovery loop for ECA worlds with persistent
  `WorldRecord` history, dedup-gated observers, and a fixed seven-law evaluator.
- Published a 20-world empirical atlas with seven dynamic categories and
  per-law coverage across:
  `velocidad_constante`, `periodicidad`, `densidad_estable`, `tipo_unico`,
  `complejidad_alta`, `frontera_temporal`, and `temporal_scale_stability`.
- Measured one-bit IC fragility using `f_total`, `f_core`, and `f_gap`,
  separating core-regime changes from secondary-law churn.
- Identified `rule_108` as the unique stationary local period-2 oscillator
  among quiescent ECA rules under the current protocol.
- Characterized `rule_54` noise-boundary fragility and observer
  translation non-equivariance.
- Validated `frontera_temporal` as a family marker, not an intrinsically rare
  law, while distinguishing atlas-grade worlds from short-scale sweep positives.
- Published the preprint PDF and registered citation metadata in the repo.

### Repository

- Added MIT license for source code.
- Added public README, reproducibility guide, and release changelog.
- Excluded raw JSONL artifacts larger than 1 MB from git while keeping scripts
  and summary reports tracked.
